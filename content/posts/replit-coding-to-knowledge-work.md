去年这个时候，大家还在讨论「AI 能不能写出能跑的代码」。今年，这个问题基本有答案了——能。而且不只是能跑，是能直接上生产环境的那种能。

但有意思的是，当代码这件事被 AI 搞定之后，所有做 Coding Agent 的公司都在同时做另一件事——**往上走**。

过去两个月，Pi 变成了 OpenClaw，Claude Code 推出了 Cowork 模式，Notion 发布了 Custom Agents。Replit 更直接，直接把整个办公套件都做了一遍：Canvas、Apps、Sites、Slides、Videos。除了代码编辑器，他们几乎把知识工作者每天要做的事都覆盖了。

这不是巧合。当代码不再是瓶颈，瓶颈就变成了代码之外的那些事。

---

## 代码已经不是瓶颈了

Replit 的 CEO Amjad Masad 在最近的采访里说了一句话：

> "Software engineering is approximately solved."

翻译过来就是：**写代码这件事，基本上已经被 AI 搞定了。**

不是说程序员要失业——而是说，当你有一个需求，从需求到代码的这段路，AI 已经能走得很顺了。剩下的问题不是「代码怎么写」，而是「需求从哪来」「做出来的东西给谁用」「怎么让更多人用上」。

这些问题的答案，都不在代码编辑器里。

---

## 从 Coding Agent 到 Knowledge Work Agent

Replit Agent 4 发布的时候，他们没有强调「代码写得更好了」，而是强调了一堆听起来跟编程没什么关系的东西。

**Canvas** 是类似 Notion 的文档编辑器，但能直接调用 Agent 生成内容。**Apps** 把代码变成可交互的应用，不需要你懂前端。**Sites** 一键生成网站，包括 SEO 优化和响应式布局。**Slides** 做演示文稿，AI 帮你排版和配图。**Videos** 生成视频内容，从脚本到剪辑。

这些东西的共同点是什么？它们都是知识工作者每天要做的事。写文档、做 PPT、搭网站、剪视频——这些活以前要么外包，要么自己硬着头皮学工具。现在 Replit 说：你告诉我你要什么，我帮你做出来。

更关键的是，这些功能不是孤立的。你在 Canvas 里写需求文档，Agent 能直接根据文档生成代码；代码跑起来之后，Agent 能帮你把它变成一个 App；App 做好了，Agent 能帮你生成一个介绍它的网站和演示视频。整个流程是连贯的。

这才是 Replit 真正想做的事——不是一个写代码的工具，而是一个「从想法到交付」的完整平台。

## 为什么是现在？

这个转向不是 Replit 拍脑袋想出来的。背后有两个技术前提。

第一个是模型的推理能力跨过了某个临界点。2025 年底之前，大部分 Coding Agent 还在跟「理解复杂需求」这件事较劲。你给它一个模糊的描述,它要么理解错，要么做出来的东西不能用。2025 年底之后，新一代模型出来，情况变了。它们不只是能写代码，而是能理解你想做什么，然后自己规划怎么做。这个能力的提升，让 Agent 从「执行工具」变成了「协作伙伴」。

第二个是多模态能力的成熟。以前 AI 只能处理文字和代码，现在它能看图、能生成图、能剪视频、能做网页。这意味着它不只能帮你写代码，还能帮你做代码之外的所有事——设计、文案、演示、推广。当这些能力都具备了，从 Coding Agent 到 Knowledge Work Agent 的跨越就是自然而然的事。

## 这对独立开发者意味着什么？

如果你是独立开发者，这个趋势对你来说是好消息。

以前做一个产品，你要会写代码、会设计、会写文案、会做推广。现在你只需要会一件事——**想清楚你要做什么**。剩下的事，AI 都能帮你做。

但这也意味着竞争会更激烈。当技术门槛降低，能做出产品的人会变多。到那时候，真正的竞争点不是「你能不能做出来」，而是「你做的东西有没有人要」。

这就回到了最开始的问题：需求从哪来？做出来的东西给谁用？怎么让更多人用上？

这些问题，AI 帮不了你。你得自己想清楚。

## 常见问题

Q: Coding Agent 会取代程序员吗？

不会。但它会改变程序员的工作内容。以前程序员的时间主要花在写代码上，以后会更多花在理解需求、设计架构、做决策上。写代码这件事会变成最不重要的那部分。

Q: 独立开发者现在应该学什么？

学怎么找需求、怎么验证想法、怎么做推广。技术能力仍然重要，但不再是唯一重要的事。当 AI 能帮你写代码，你的价值就在于你能想到别人想不到的点子，能找到别人找不到的用户。

Q: Replit 这些功能对普通人有用吗？

有用。如果你不是程序员，但想做个网站、做个小工具、做个演示文稿，Replit 这类平台能帮你省掉学习成本。你只需要说清楚你要什么，AI 帮你做出来。
