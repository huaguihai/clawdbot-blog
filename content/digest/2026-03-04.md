**1. Claude Code 新增语音模式：用嘴写代码成为可能**

来源：TechCrunch AI

Anthropic 为其终端编码工具 Claude Code 推出语音模式功能。开发者现在可以通过语音与 Claude Code 交互，描述需求、审查代码、执行命令。

这个更新对特定场景很有价值：比如在调试时双手忙于操作键盘，或者在白板前讨论架构时想快速验证想法。语音模式降低了与 AI 编码助手交互的门槛，尤其适合有无障碍需求的开发者。

关键数据：Claude Code 新功能，支持语音交互

🔗 [原文链接](https://techcrunch.com/2026/03/03/claude-code-rolls-out-a-voice-mode-capability/)

**2. Anthropic 升级 Claude 记忆功能：免费用户也能用，支持从其他 AI 迁移**

来源：The Verge AI

Anthropic 宣布将 Claude 的记忆功能下放到免费用户，同时推出专门的工具帮助用户从其他聊天机器人导入对话历史和偏好设置。

记忆功能此前是付费用户专属，现在免费开放。更值得关注的是"迁移工具"——用户可以将 ChatGPT 的对话历史和偏好导入 Claude，配合近期 Claude 在应用商店排名上升的势头，Anthropic 正在系统性地降低用户切换成本。

关键数据：记忆功能免费开放、新增迁移工具

🔗 [原文链接](https://www.theverge.com/ai-artificial-intelligence/887885/anthropic-claude-memory-upgrades-importing)
